# SleepLock - K70 Pro 专注锁机应用

📱 帮助用户建立规律作息的安卓锁机应用

## 核心功能

- ⏰ **定时锁屏** - 23:40 自动锁定，04:00 自动解锁
- 📵 **应用白名单** - 仅允许电话、短信等必要应用
- 😴 **睡眠监测** - 自动记录睡眠时长，生成统计报告
- 💰 **额度系统** - 提前睡觉赚额度，重启 3 次解锁（清零额度）
- 📅 **节假日调整** - 周五六/节假日自动推迟 1 小时

## 技术栈

- **语言**: Kotlin 1.9+
- **UI**: Jetpack Compose
- **架构**: MVVM + Clean Architecture
- **依赖注入**: Hilt
- **数据库**: Room
- **后台任务**: WorkManager
